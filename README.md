# ER Stock Calculator

Internal stock sending calculator for ER Operations (BLR / KOL / DEL).

## Deploy to Render

1. Push this folder to a GitHub repo
2. Go to render.com → New → Static Site
3. Connect your GitHub repo
4. Set:
   - **Build Command:** `npm install && npm run build`
   - **Publish Directory:** `dist`
5. Click Deploy

## Local Development

```bash
npm install
npm run dev
```
